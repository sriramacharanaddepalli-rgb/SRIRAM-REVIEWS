# Sriram Reviews 🎬

A professional cinema archive and news portal built by an aspiring filmmaker and scriptwriter. 

## 🚀 Features
- **Dual Feed System:** Toggle between technical movie reviews and the latest industry news.
- **Persistent Storage:** Content is saved locally using Browser LocalStorage.
- **Cinematic UI:** Dark-themed responsive design optimized for film enthusiasts.
- **Mobile Ready:** Built with a "Mobile-First" approach for on-the-go reading.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure (93-line optimized version).
- **CSS3:** Custom properties and flexbox layout.
- **JavaScript:** ES6 filtering and storage logic.
